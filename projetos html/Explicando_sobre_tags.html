TAG:H1 - H6

voltar

A tag <H1> ao <H6> no HTML é usada para definir o nivel de um cabeçalho, o qual H1 é o mais alto em termos de hierarquia entre os cabeçalhos e o H6 o menor. Cabeçalhos HTML são usados para estruturar o conteúdo de uma página web, fornecendo diferentes níveis de título e sub-título.


TAG: <P>

voltar

A Tag <p> no HTML é usada para definir parágrafos. Ela é uma das tags mais básicas para estruturar o conteúdo textual em uma página web. Quando você utiliza a tag <p>, o navegador renderiza o texto dentro dela como um parágrafo, geralmente adicionando algum espaçamento antes e depois do texto para tornar visualmente claro que se trata de um novo parágrafo.


TAG: <MARK>

voltar

A Tag <mark> no HTML é usada para destacar partes do texto, geralmente com o objetivo de chamar a atenção do usuário para informações específicas. Quando você envolve um pedaço de texto com a tag <mark>, o navegador geralmente o destaca com uma cor de fundo amarela padrão, embora essa aparência possa ser alterada por meio de estilos CSS.

TAG: <SMALL>

voltar

A Tag: <small> é uma tag HTML que é usada para reduzir o tamanho do texto e indicar que o texto contido dentro dela é de tamanho menor ou de importância secundária. Geralmente, navegadores exibem o texto dentro de <small> em um tamanho de fonte menor do que o texto circundante.


TAG: <i>

voltar

A Tag <i> no HTML é tradicionalmente usada para indicar que o texto contido dentro dela deve ser exibido em itálico. No entanto, é importante notar que, no contexto mais moderno e semântico do HTML5, a tag <i> por si só não transmite um significado semântico específico. Se você estiver usando <i> para enfatizar o texto, seria mais apropriado usar a tag <em> (ênfase) ou <strong> (forte ênfase) dependendo do contexto semântico. Isso ajuda a transmitir ao navegador e a outros agentes de usuário a intenção semântica do texto, além de permitir uma melhor acessibilidade.


TAG: <U>

voltar

A Tag <u> no HTML é usada para sublinhar o texto. Ela era originalmente usada para indicar texto sublinhado, mas, assim como a tag <i> para itálico, a tag <u> não é mais considerada semanticamente apropriada para representar significado. No contexto do HTML5, recomenda-se o uso de CSS para estilizar o texto, incluindo a aplicação de sublinhado.


TAG: <STRONG>

voltar

A tag <strong> no HTML é usada para indicar que o texto contido dentro dela é semanticamente importante ou de forte ênfase. O texto marcado com <strong> é geralmente exibido em negrito pelos navegadores, embora a principal função da tag seja transmitir significado semântico, não apenas estilo visual.


TAG: <OL>

voltar

A Tag <ol> no HTML é usada para criar listas ordenadas, ou seja, listas em que a ordem dos itens é significativa. Cada item em uma lista ordenada é marcado por um número, geralmente começando do número 1.


TAG: <UL>

voltar

A Tag <ul> no HTML é usada para criar listas não ordenadas, onde a ordem dos itens não é significativa. Os itens em uma lista não ordenada são geralmente marcados por marcadores de lista, como pontos (•), círculos (◦), ou quadrados (▪)


TAG: <LI>

voltar

A tag <li> no HTML é usada para definir cada item de uma lista. Essa tag é usada dentro de elementos <ul> (listas não ordenadas) ou <ol> (listas ordenadas) para indicar cada item específico na lista.


TAG: <A>

voltar

No HTML é usada para criar links, permitindo que os usuários naveguem de uma página para outra ou dentro da mesma página. O atributo "HREF" é usado para especificar o URL para o qual o link aponta.


TAG: <HR>

voltar

A tag <hr> no HTML é usada para criar uma linha horizontal, indicando uma mudança de tema ou separação entre elementos em uma página da web. O <hr> é um elemento vazio, o que significa que não possui uma tag de fechamento correspondente.


TAG: <SUB>

voltar

A tag <sub> no HTML é usada para representar o texto que deve ser exibido como subscrito. O texto subscrito geralmente aparece abaixo da linha de base normal e é frequentemente usado para representar fórmulas químicas, expressões matemáticas, datas, marcas de registro e outras notações onde o texto deve ser exibido ligeiramente abaixo do texto adjacente.


TAG: <SUP>

voltar

A tag <sup> no HTML é usada para representar texto sobrescrito, que é exibido acima da linha de base normal. Essa tag é frequentemente usada para expressar expoentes em fórmulas matemáticas, datas ordinais (como 1º para "primeiro"), ou para qualquer texto que deva aparecer acima da linha de texto padrão.


TAG: <BLOCKQUOTE>

voltar

A tag <blockquote> no HTML é usada para citar um bloco de texto de outra fonte. Ela é comumente usada para citar parágrafos ou seções longas de texto provenientes de uma fonte externa, como um livro, artigo ou página da web.
